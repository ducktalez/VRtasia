# VRtasia

Dieses Repository ist der gemeinsame Startpunkt, um mit Freunden ein VR-Spiel zu entwickeln.

## Ziel
- Zusammenarbeit an einem VR-Spiel in einem zentralen Repository
- Nachvollziehbare Versionierung von Ideen, Assets und Code

## Nächste Schritte
1. Game-Engine und Tech-Stack festlegen
2. Projektstruktur (z. B. `docs/`, `assets/`, `src/`) definieren
3. Aufgaben als Issues planen und im Team verteilen
